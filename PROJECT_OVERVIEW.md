# YC365 商户管理系统 - 项目概览

## 项目信息

- **项目名称**: YC365 商户管理系统 (Merchant Client)
- **版本**: 1.0.0
- **开发时间**: 2025年12月
- **技术栈**: React 18 + TypeScript + Vite + Ant Design 5

## 项目背景

YC365 作为预测服务提供商，为第三方提供预测服务是重要的业务拓展方向。商户管理系统使商户能够管理用户、资产、权限和安全等核心业务。

## 核心价值

1. **用户管理**: 商户可以查看和管理自己的用户，了解用户订单和消费情况
2. **资产管理**: 实时查看和管理商户资产，支持充值和提现
3. **权限控制**: 灵活的角色权限体系，支持系统管理员和业务管理员
4. **API 管理**: 便捷的 API Key 管理，支持流量统计
5. **积分系统**: 商户可以查看和分配积分，激励用户使用

## 系统架构

### 技术架构

```
┌─────────────────────────────────────────┐
│           浏览器 (Browser)              │
└─────────────────┬───────────────────────┘
                  │
┌─────────────────▼───────────────────────┐
│      前端应用 (React + TypeScript)      │
│  ┌─────────────────────────────────┐   │
│  │  UI Layer (Ant Design)          │   │
│  ├─────────────────────────────────┤   │
│  │  Business Logic (Pages)         │   │
│  ├─────────────────────────────────┤   │
│  │  State Management (Zustand)     │   │
│  ├─────────────────────────────────┤   │
│  │  API Layer (Axios)              │   │
│  └─────────────────────────────────┘   │
└─────────────────┬───────────────────────┘
                  │ HTTP/HTTPS
┌─────────────────▼───────────────────────┐
│      后端 API (Backend Services)        │
└─────────────────────────────────────────┘
```

### 数据流

```
用户操作 → 页面组件 → API 调用 → 后端处理 → 响应数据 → 状态更新 → 界面刷新
```

## 功能模块

### 1. 认证模块

- **登录**: 支持商户域名、账号、角色、密码登录
- **Token 管理**: JWT Token 认证机制
- **路由守卫**: 基于角色的路由访问控制
- **会话管理**: 自动刷新和过期处理

### 2. 系统管理员模块

#### 2.1 工作台
- 业务管理员统计
- 资产概览
- 数据趋势分析

#### 2.2 业务管理员管理
- 创建业务管理员（账号、密码、权限）
- 重置密码
- 权限管理（6种权限）
- 启用/禁用账号

#### 2.3 资产管理
- 充值功能（连接钱包）
- 提现功能（连接钱包）
- 交易记录查询
- 交易状态跟踪

### 3. 业务管理员模块

#### 3.1 工作台
- 资产统计
- 用户统计
- API Key 统计
- 积分统计

#### 3.2 Sign 服务配置
- 添加/编辑/删除配置
- 测试服务连接
- 激活配置
- 确保至少一个可用配置

#### 3.3 资产查询
- 总资产、已使用、剩余资产
- 使用率可视化
- 钱包地址查看

#### 3.4 用户管理
- 用户列表查询（支持搜索）
- 用户详情查看
- 用户订单查询
- 数据统计

#### 3.5 API Key 管理
- 创建 API Key（最多5个）
- 查看和复制 Key
- 重置 Key
- 启用/禁用 Key
- 流量统计
- 删除 Key

#### 3.6 积分管理
- 查看积分信息
- 分配积分给用户
- 分配记录查询
- 余额检查
