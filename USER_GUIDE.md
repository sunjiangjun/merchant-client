## 系统简介

YC365 商户管理系统是一个面向商户的预测服务管理平台，提供用户管理、资产管理、权限管理、API Key 管理等功能。

### 角色说明

- **系统管理员**: 负责创建和管理业务管理员，管理商户资产
- **业务管理员**: 负责日常业务操作，包括用户管理、API Key 管理、积分管理等

## 快速开始

### 登录系统

1. 访问系统登录页面
2. 输入以下信息：
   - **商户域名**: 您的商户域名（由 YC365 提供）
   - **账号**: 您的登录账号
   - **角色**: 选择系统管理员或业务管理员
   - **密码**: 您的登录密码
3. 点击"登录"按钮

### 注册说明

商户账号注册需要联系 YC365，由 YC365 为商户注册系统管理员账号。

联系方式：
- 邮箱: support@yc365.com
- 电话: 400-xxx-xxxx

## 系统管理员使用指南

### 1. 工作台

登录后进入工作台，可以查看：
- 业务管理员总数
- 活跃管理员数量
- 商户总资产
- 月度资产变化

### 2. 业务管理员管理

#### 创建业务管理员

1. 进入"业务管理员"页面
2. 点击"创建业务管理员"按钮
3. 填写以下信息：
   - **账号**: 业务管理员登录账号
   - **初始密码**: 设置初始登录密码
   - **权限**: 选择该管理员的操作权限
     - 查看资产
     - 查看用户
     - 管理 API Key
     - 查看积分
     - 分配积分
     - 配置 Sign 服务
4. 点击"确定"完成创建

#### 重置业务管理员密码

1. 在业务管理员列表中找到目标管理员
2. 点击"重置密码"按钮
3. 输入新密码并确认
4. 点击"确定"完成重置

#### 更新业务管理员权限

1. 在业务管理员列表中找到目标管理员
2. 点击"权限"按钮
3. 勾选或取消勾选权限项
4. 点击"确定"保存更改

#### 启用/禁用业务管理员

1. 在业务管理员列表中找到目标管理员
2. 点击"禁用"或"启用"按钮
3. 确认操作

### 3. 资产管理

#### 充值

1. 进入"资产管理"页面
2. 点击"充值"按钮
3. 输入以下信息：
   - **钱包地址**: 您的钱包地址（42位以太坊地址）
   - **金额**: 充值金额（USDT）
4. 点击"确定"提交充值请求
5. 连接钱包并确认交易

**注意**: 充值操作需要连接钱包并完成链上交易确认。

#### 提现

1. 进入"资产管理"页面
2. 切换到"提现记录"标签
3. 点击"提现"按钮
4. 输入提现信息：
   - **钱包地址**: 接收钱包地址
   - **金额**: 提现金额（USDT）
5. 点击"确定"提交提现请求

**注意**: 提现操作需要连接钱包并完成链上交易确认。

#### 查看交易记录

- **充值记录**: 查看所有充值交易记录
- **提现记录**: 查看所有提现交易记录

交易状态说明：
- **处理中**: 交易正在处理中
- **已完成**: 交易已完成
- **失败**: 交易失败

## 业务管理员使用指南

### 1. 工作台

查看商户业务数据概览：
- 总资产、已使用资产、剩余资产
- 用户总数
- API Keys 数量
- 积分余额

### 2. Sign 服务配置

Sign 服务用于为 API 请求提供签名服务。

#### 添加 Sign 服务配置

1. 进入"Sign 服务配置"页面
2. 点击"添加配置"按钮
3. 输入以下信息：
   - **Sign 服务 URL**: Sign 服务的访问地址
   - **描述**: 配置说明
4. 点击"确定"添加配置

#### 测试 Sign 服务连接

1. 在配置列表中找到目标配置
2. 点击"测试"按钮
3. 系统会验证 Sign 服务是否可用

#### 激活配置

1. 在配置列表中找到目标配置
2. 点击"激活"按钮
3. 该配置将成为当前使用的 Sign 服务

**注意**: 
- 必须确保有且仅有一个 Sign URL 是可用的
- 删除配置时，如果是唯一激活的配置，则无法删除

### 3. 资产查询

查看商户资产情况：
- **总资产**: 商户账户总资产
- **已使用资产**: 已消耗的资产
- **剩余资产**: 可用资产余额
- **资产使用率**: 可视化展示资产使用情况
- **钱包地址**: 商户钱包地址（如已绑定）

### 4. 用户管理

#### 查询用户列表

1. 进入"用户管理"页面
2. 可以查看商户下所有用户信息：
   - 用户名、邮箱、手机号
   - 订单数、总消费
   - 注册时间、最后活跃时间

#### 搜索用户

在搜索框中输入用户名、邮箱或手机号进行搜索。

#### 查看用户详情

1. 在用户列表中点击"详情"按钮
2. 查看用户完整信息

#### 查看用户订单

1. 在用户列表中点击"订单"按钮
2. 查看该用户的所有订单记录

### 5. API Key 管理

API Key 用于调用 YC365 预测服务 API。

#### 创建 API Key

1. 进入"API Key 管理"页面
2. 点击"创建 API Key"按钮
3. 输入 API Key 名称
4. 点击"确定"生成 API Key
5. **重要**: 请立即复制并保存 API Key，后续无法再次查看完整 Key

**限制**: 每个商户最多可以创建 5 个 API Key。

#### 查看 API Key

- 在列表中可以查看所有 API Key
- Key 会部分隐藏显示，点击复制图标可复制完整 Key

#### 重置 API Key

1. 在列表中找到目标 API Key
2. 点击"重置"按钮
3. 确认操作
4. 系统会生成新的 Key，旧 Key 将失效

**注意**: 重置后，使用旧 Key 的应用将无法访问 API，请及时更新。

#### 启用/禁用 API Key

1. 在列表中找到目标 API Key
2. 点击"启用"或"禁用"按钮
3. 禁用后该 Key 将无法访问 API

#### 查看流量统计

1. 在列表中找到目标 API Key
2. 点击"流量"按钮
3. 查看该 Key 的流量使用情况

#### 删除 API Key

1. 在列表中找到目标 API Key
2. 点击"删除"按钮
3. 确认操作

### 6. 积分管理

#### 查看积分信息

- **总积分**: 商户获得的总积分
- **已使用积分**: 已分配给用户的积分
- **剩余积分**: 可用于分配的积分

#### 分配积分给用户

1. 进入"积分管理"页面
2. 点击"分配积分"按钮
3. 选择目标用户
4. 输入分配的积分数量
5. 输入备注（可选）
6. 点击"确定"完成分配

**注意**: 分配的积分不能超过剩余积分。

#### 查看分配记录

在"积分分配记录"中可以查看所有积分分配历史：
- 用户信息
- 分配积分数
- 分配人
- 分配时间
- 备注

## 常见问题

### Q: 忘记密码怎么办？

**A**: 
- 业务管理员：联系系统管理员重置密码
- 系统管理员：联系 YC365 技术支持

### Q: 为什么无法删除 Sign 服务配置？

**A**: 系统要求至少保留一个可用的 Sign 服务配置。如果要删除当前唯一激活的配置，请先添加并激活新的配置。

### Q: API Key 丢失了怎么办？

**A**: 出于安全考虑，系统不会存储完整的 API Key。如果丢失，请重置该 Key 生成新的 Key。

### Q: 充值/提现需要多长时间？

**A**: 
- 充值通常需要 3-10 分钟完成链上确认
- 提现需要人工审核，通常在 24 小时内完成

### Q: 可以创建多少个 API Key？

**A**: 每个商户最多可以创建 5 个 API Key。

### Q: 积分有有效期吗？

**A**: 积分没有有效期限制，但分配给用户后无法收回。

### Q: 如何查看 API 使用情况？

**A**: 在"API Key 管理"页面，点击每个 Key 的"流量"按钮可以查看使用统计。

### Q: 业务管理员可以修改自己的权限吗？

**A**: 不可以。业务管理员的权限只能由系统管理员修改。
